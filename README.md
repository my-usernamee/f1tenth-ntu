# NTU DeepSpeed Website

A responsive static website for NTU DeepSpeed, an F1TENTH / RoboRacer style
student autonomous racing team at NTU.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive visual design
- `script.js` - mobile navigation, component highlighting, and GSAP motion
- `assets/deepspeed-logo.png` - team logo

## Host on GitHub Pages

1. Create a new GitHub repository, for example `f1tenth-ntu` or `ntu-deepspeed`.
2. Add these files to the root of the repository.
3. Commit and push to the `main` branch.
4. On GitHub, open the repository and go to `Settings` > `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main` and folder `/(root)`, then save.

Your site should be available at:

```text
https://<your-github-username>.github.io/<repo-name>/
```

If you create a repository named `<your-github-username>.github.io`, the site will be available at:

```text
https://<your-github-username>.github.io/
```

Before publishing widely:

- Replace `your-email@ntu.edu.sg` with the club email, form link, or social link.
- Replace placeholder result numbers with real achievements.
- Replace placeholder photo blocks with team, car, track, and event photos.
- Update the car component labels with the exact hardware used by the team.
